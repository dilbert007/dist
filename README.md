# Mango binary repo 

## Application 

This is a small cross platform (Mac, W10 and Linux) to play videos and photos easily.

It will open a player in full screen on the second monitor (or TV) and you can show the photos or videos there.

### Mango

Load all photos and videos from a folder or add a signle item.

**Help**

  * Control + F 
    Load all the images or videos from a folder.
  * Control + I 
    Load one image or video.
  * Control + R 
    Clear all.
  * Control + Q 
    Quit the applications.
  * Z 
    Set the image to natural aspect ratio.
  * X 
    Set the image to stretch aspect ratio.
  * A, W, S, D 
    Rotate the image.
  * Escape key
    Stops the image or video.
  * Space bar
    Switch between hide and show image in the player.


**New Features**

 * Drag and drop images or videos to add them to the list.
 * Aspect ratio fixed in the player.
 * Deb package created
